Hobbies

In my free time, I like to learn new things; they don't have to be computer related. Every summer, I embark on a project. Some projects from previous years:

  • When I moved into my new house, I built on-prem camera security system for my home using Frigate. It utilizes a Google Coral TPU for real-time computer vision, without any dependency on cloud services. I currently use it to check whether my cat, Casper, is eating his food while I'm not at home.
  • During the pandemic, I set up a home automation system using Home Assistant. My current setup is a Zigbee network (for lights and sensors), ESPHome sensors, a MQTT server, and a PostgreSQL database running on a Docker stack.
  • I 3D print objects that I designed in Fusion 360. It's a very satisfying feeling to physically hold something that you ideated on a computer.
  • I've also set up a Dropbox and Google Photos replacement with Nextcloud. Recently, I implemented a neural net to recognize and group faces in my photo collection, which runs offline from the cloud (though nothing can ever beat Google's high quality data).
  • I used RMarkdown (finally) to publish some of the material I wrote for an experimental statistics course at TJ on this website.
  • I've accepted my fate as a product of the suburbs and can talk about lawncare from the hours I spend doing it.
