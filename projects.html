Projects

Intro to R for High School Students

A comprehensive textbook that introduces basic Statistics concepts and shows how to execute them in R. Written for students who are in the midst of taking a Introductory Statistics class.

Gives students examples of how to create data visualizations with base R Graphics and ggplot, run one-sample and two-sample t-tests, and one-way ANOVA. Includes Shiny apps for coding practice and code checking.

Home Assistant Automation System

A local home autuomation server using Home Assistant, controlling lights, TV, climate, cleaning functions, and several Alexa speakers. Runs on a Zigbee and MQTT network, proxied through a Cloudflare Zero-Trust tunnel.

Pulls Geolocation data from phones, weather data from Tomorrow.io, presence and motion sensor detection data, and other sensor data into a local PostgreSQL database.

Lights change color temperature based on time of day and weather condition. Lights and HVAC automatically turn off based on presence detection. Connected to Alexa and Homekit proxied through local servers and Cloudflare.

Personal Cloud Sever with Nextcloud

Created a personal Dropbox-like server using Nextcloud, running on Network Attached Storage at home. Runs on Docker, proxied through Cloudflare Zero-Trust, with Redis server for memory caching and PostgreSQL for backend.
